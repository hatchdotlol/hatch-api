# hatch API

This is the repository for the API used by hatch.lol.

Public instance at https://api.hatch.lol, see https://hatch.lol for more info.

## Requirements

You need a running instance of MinIO object storage, and SQLite3 drivers and [ImageMagick](https://imagemagick.org/script/download.php) installed locally.

(for design reference, the rust api source is over on [this commit](https://github.com/hatchdotlol/hatch-api/tree/9d1408fe35e4e638e6d7d6cc3794258e7c80f679))
